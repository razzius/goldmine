An easy tool to trim your libraries

GoldMine is a command line tool that uses a coverage report to determine what parts of your libraries aren't being used and takes them out so they don't slow down your web pages

Try it out now!

Install

GoldMine is on npm so it is fast and easy to install!
>sudo npm install -g goldmine
