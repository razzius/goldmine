Easily trim external libraries

GoldMine is a command line tool that uses a coverage report to determine what parts of your libraries aren't being used and takes them out so they don't slow down your web pages

Try it out now!

Install

GoldMine is on npm so it is fast and easy to install!
$ sudo npm install -g goldmine

Command Line Functionality

GoldMine can be run straignt from your terminal with the following arguments
$ goldmine -t tests.js -c code.js -l library.js

Instrumenting

GoldMine instruments with istanbul, but utilizes its own coverage framework to find unnecessary code

Remove the code

Finally GoldMine trims the external libraries so the extra stuff you don't use doesn't way you down

Did you know?

The lower bound of JQuery coverage is only 20%, if you only implement one method you could only be using a quarter of the library
This idea can be applicable to much larger libraries as well! (of course JQuery is a small example)
